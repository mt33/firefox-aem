* Touch UI
  *   Home
  *   Pages
  *   Templates
  *   XFs
  *   Assets
  *   CFs
  *   CF Models
* Classic UI
  *   Welcome
  *   Replication agents
  *   Activate Tree
  *   Useradmin
  *   Miscadmin
  * CRX areas
  *   CRXDE
  *   Packages
* OSGI areas
  *   Configs
  *   Services
  *   Bundles
  *   Script cache
  *   FSClassLoader
